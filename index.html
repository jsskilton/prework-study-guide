HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • < link rel="stylesheet" href="./assets/style.css" >: links to the file style.css in the same directory
  • In the preceding statement, a < link > element creates the connection to the external file.
  • The rel attribute specifies the relationship between the current document and the linked document resource.
  • The href attribute specifies the location (URL) of the external resource. The href attribute is using a relative path.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: commits all files in the working branch
  • git commit -m "add description for github": commits the changes
  • git pull origin main: pull the base branch into your feature branch
  • git push origin feature/add-html: push up the changes in feature/add-html

Push to Git

  1. Navigate to branch from the command line
  2. git add - A
  3. git commit -m "add description"
  4. git pull origin main
  5. git push origin specific/branch
  6. In GitHub, go to repository click 'pull requests' > 'new pull request'
  7. Make sure the base branch is main and comparison branch is specific/branch
  8. Click the 'create pull request' button
  9. Merge the comparison branch into teh base branch by selecting the 'merge pull request' button
  10. Update the local main branch via: git checkout main
  11. Pull the remote main branch into the local main branch via: git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
